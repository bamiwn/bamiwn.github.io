Teknologi yang Saya Kuasai adalah..

Basic Website

Mencakup: HTML, CSS, Bootstrap, dan Javascript. HTML sebagai kerangka, CSS sebagai tampilan, Bootstrap sebagai Framework UI website, dan JS sebagai interaktifitas.

Website Dinamis

Mencakup: MySQL & PHP. MySQL sebagai tempat penyimpanan data, dan PHP sebagai logic programming untuk menangani semua proses data.

Frontend Lanjutan

Mencakup: Tailwind CSS, VueJS, NuxtJS, & NPM (Sedang belajar)

Backend Lanjutan

Mencakup: Object-Oriented-Programming PHP, Laravel (Framework). OOP PHP sebagai konsep dasar framework agar mengetahui alur kerjanya. Sedangkan Laravel sebagai framework Backend PHP Website.

Eksplorer API

Mencakup: REST API, JSON. REST API sebagai API pihak ke-3 agar website lebih berfungsionalitas serta SANGAT PERLU untuk bisnis. Sedangkan JSON, sebagai pertukaran data lewat HTTP.

Fitur & Analisis Website

Pernah membuat: Login multiuser, OAuth (Socialite), CRUD, Multiselect (LiveWire), Auth (Fortify & Jetstream), Desain & Analisis Database.
